# SDET/AQA Handbook

Practical knowledge base for experienced SDET/AQA engineers: playbooks, templates, checklists, and Q&A.

## Value Proposition

- Fast onboarding to core testing domains (API, UI, CI/CD, System Design, AI in QA).
- Reusable practical assets for daily work.
- Structured material for increasing qualification of SDET/AQA engineers (Middle/Senior/Lead).

## How To Use This Repository

1. Start with `INDEX.md` for topic map and learning paths.
2. Pick one domain and review core materials.
3. Apply checklists/templates in real project tasks.
4. Revisit interview tracks for focused preparation.
