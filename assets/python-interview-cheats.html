Python Interview Cheats

Portable HTML reference for interview preparation and screenshot cards.

Runtime Model

Dynamic typing, strong typing, GIL in CPython, PyPy as alternative runtime.

Data Structures

Know mutability, hashing, and practical use of list/dict/set/tuple/deque.

Complexity

Dict/set lookup average O(1), list search O(n), list index access O(1).

Concurrency

Threads for IO-bound, processes for CPU-bound, asyncio for high-concurrency IO.

Generators

Use yield for lazy iteration, memory-efficient streams, and fixture-like flows.

Context Managers

with / __enter__ / __exit__ for deterministic resource cleanup.
